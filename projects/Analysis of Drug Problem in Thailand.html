Data Analysis Project: Sales Data Forecasting

Project Description

This project involves forecasting future sales for a retail company using historical sales data. I applied time series analysis and regression models.

Technologies Used

  • Python (Pandas, NumPy, Matplotlib)
  • SQL
  • Excel (Pivot Tables)

Key Insights & Results

Based on the analysis, we were able to predict sales with 90% accuracy, helping the company optimize inventory levels.
